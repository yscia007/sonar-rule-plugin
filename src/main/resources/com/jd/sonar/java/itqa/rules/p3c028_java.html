ArrayList的subList结果不可强转成ArrayList，否则会抛出ClassCastException 异常，

即 java.util.RandomAccessSubList cannot be cast to java.util.ArrayList.

说明：subList 返回的是 ArrayList 的内部类 SubList，并不是 ArrayList ，而是ArrayList 的一个视图，对于 SubList 子列表的所有操作最终会反映到原列表上。

错误示例

  
    public class Foo {
            private void method(long aLong) {
               List<String> list = new ArrayList<String>();
               list.add("22");
               List<String> test = (ArrayList<String>) list.subList(0, 1);   // Noncompliant
             }
        }
