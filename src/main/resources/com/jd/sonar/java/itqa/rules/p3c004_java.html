代码中的命名均不能以下划线或美元符号开始，也不能以下划线或美元符号结束。

错误示例

  
      public class Foo {
    private String $1;
    private String _1;
    private String ee$;
    private String ee_;
    public void $sayHello(){}
    public void _sayHello(){}
    public void sayHello$(){}
    public void sayHello_(){}
}
  
     
代码中有8处违规，违规行：2,3,4,5,6,7,8,9。

正确示例

public class AvoidStartWithDollarAndUnderLineNamingRuleTest {
    private String s;
    public void sayHello(){}
}
