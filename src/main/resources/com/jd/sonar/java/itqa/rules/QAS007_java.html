Escape characters in split should be paid special attention.

小心String.split(String regex)

Noncompliant Code Example

  
    "a.ab.abc".split(".");// 结果为[]
    "a|ab|abc".split("|");// 结果为["a", "|", "a", "b", "|", "a", "b", "c"]

compliant Code Example

  
    "a.ab.abc".split("\\.");// 结果为["a", "ab", "abc"]
    "a|ab|abc".split("\\|");// 结果为["a", "ab", "abc"]
