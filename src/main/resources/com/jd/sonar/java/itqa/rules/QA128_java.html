通过扫描spring配置文件获取Spring项目jsf配置文件中的jsf生产者接口

正确示例

spring配置文件jsf生产者示例：

    
        <jsf:provider id="jsfPriceProService" interface="com.jd.project.server.JsfPriceProService"  alias="${proprice.jsf.alias}" ref="jsfPriceProServiceImpl" server="priceprojsf">
        </jsf:provider>

上述配置文件中jsf生产者接口为：com.jd.project.server.JsfPriceProService
